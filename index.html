What is Brm5 Mapper?

BRM5 Mapper is a free server based planning tool for both raids and stealth missions. This tool includes features like:

  • Marking Locations
  • Built in text-chat
  • Code Based Sessions(Which means you can either create or join a session based on a code)

    • Why should I use BRM5 Mapper

    If you are looking for a co-op planning tool, then use BRM5 Mapper. BRM5 Mapper also recieves multiple updates to better the users experience with BRM5 Mapper. Here is some extra information:

  • Saved data(About 5 minutes of inactivity will end the session to save bandwidth)
  • No extra api(So nothing to do with downloading extra packages from the internet)

    • Public Server

    We have a public server id called '0000'. Type 0000 into the code area and you can join other planning.
